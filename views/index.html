Request Header Parser Microservice

How it works

You get the IP address, preferred languages and system info for your device by adding /api/whoami to the address in the address bar.

Example Usage

Just click this link to do it for you!

[base url]/api/whoami

Example Output

{"ipaddress":"159.20.14.100","language":"en-US,en;q=0.5", "software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
